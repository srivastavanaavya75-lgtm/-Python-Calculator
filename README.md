# Simple Calculator (Python)

This is a basic command-line calculator built using Python. It performs arithmetic operations like addition, subtraction, multiplication, and division.

## Features
- Supports +, -, *, /
- Handles invalid input using try-except
- Prevents division by zero
- Runs continuously until user exits

## Technologies Used
- Python

## Learning Outcome
- Learned functions and modular coding
- Used loops and conditional statements
- Implemented error handling
